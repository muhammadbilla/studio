# **App Name**: Chapchap Boutique

## Core Features:

- Product Details Page: Dedicated page for each product with detailed information.
- Similar Products Section: Display a list of similar products on the product details page, determined by product category using a tool
- Enhanced Product Cards: Improved product cards with clear details and direct link to product page.
- Product Image Carousel: Implement a carousel to showcase multiple product images.
- Dynamic product loading: Automatically loads related products, eliminating the need to hardcode data

## Style Guidelines:

- Primary color: Midnight Blue (#191970) evokes trust and professionalism.
- Background color: Off-white (#F5F5F5) for clean and modern feel.
- Accent color: Orange (#FFA500) to highlight key elements like 'Acheter' buttons and promotional badges.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text. Note: currently only Google Fonts are supported.
- Code font: 'Source Code Pro' for displaying code snippets. Note: currently only Google Fonts are supported.
- Use Font Awesome icons for categories and interactive elements.
- Implement a grid layout for product listings.
- Use subtle transitions on hover for product cards.
